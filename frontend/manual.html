// Introduction

What This Studio Does

LEVRAM Studios is an end-to-end AI film pipeline. You write an idea, the system generates a full cinematic script, creates images for every scene, animates them into video clips, adds voice-over, assembles a timeline, and exports a finished film — with you approving at every key step.

Gold steps — the AI does it automatically
Red steps — you do it manually
// Getting Started

Power-On Sequence

Every session starts the same way regardless of what you're working on:

  1. 1
    Open the studio URL
    Navigate to your Railway deployment URL. The launch screen checks all backend services. Wait for BACKEND LIVE to appear in the top-right corner before doing anything.
  2. 2
    Check the Dashboard
    Dashboard loads automatically. Check your stats (Ideas, Characters, Shots, Clips) and the Where I Left Off card. If a pipeline job was running before you closed the tab, the active job banner will reappear.
    Dashboard tab
  3. 3
    Pick up where you left off or start fresh
    Click CONTINUE → on the WIP card to jump back into an existing idea, or click NEW IDEA to start a new project.

// Step 1

Idea Vault — Write Your Idea

What you do manually

  1. 1
    Type your title
    Short name for this story (e.g. "Hulk's Death"). Used to label all generated clips and exports.
  2. 2
    Write the raw idea
    Describe the concept in natural language. The more specific you are (setting, tone, conflict, characters involved), the better the AI breaks it down. Minimum 1-2 sentences — more detail = better scenes. The textarea auto-expands as you type.
  3. 3
    Set genre, target runtime, scene length
    Genre affects the tone of image prompts and dialogue. Target runtime + scene length determines how many scenes get generated. 8 min @ 5s/scene = ~106 scenes.
  4. 4
    Click Save Idea
    Idea is stored. It appears in the saved ideas list on the right. You can edit it later with the Edit button.
// Step 2

Characters — Set Up Your Cast

Do this before generating. The pipeline uses the selected character's LoRA model for image consistency and their cloned voice for TTS. Without a character set up, images will be generic and voice-over will use a default.
  1. 1
    Go to Characters tab
    Fill in: Name, Gender, Age, Appearance description, Wardrobe, Voice description, Personality, Notes.
    Characters tab
  2. 2
    Upload reference photos (5+ for LoRA)
    Click + Upload Photos. Add at least 5 clear face images. Once you have 5+, the Train LoRA button activates. LoRA training takes 10-20 minutes in the background.
  3. 3
    Set Default Voice Profile
    Select a cloned voice from the dropdown. Voices are created in Voice Clone ↗. The selected voice is used for all TTS in the pipeline for this character.
  4. 4
    Save Character
    Click Save. The character now appears in Idea Vault's character selector when you develop a story.
// Step 3

Develop Story — AI Writes the Script

In the Idea Vault, click the idea card → then click Develop Story. GPT-4o-mini reads your raw idea and generates:

  • Title — cinematic title for the film
  • Logline — one-sentence hook
  • Act Structure — 3-act breakdown
  • Scene list — one entry per clip: description, image prompt, dialogue, emotion, reel weight
Tip: After development, scroll through the scene list in Idea Vault. You can manually edit any scene's image prompt or dialogue before generating — fixing a bad prompt here saves a re-run later.
Tip: Reel weight (1-10) determines which scenes get picked for 60s / 30s / 15s highlight reels. High-weight scenes = most cinematic moments.
// Step 4

Generate Keyframes — Images First

In the Idea Vault approve bar, click 🖼 Keyframes → Review.

Always use the two-phase flow. Keyframes are free (FLUX). Animation costs money (Kling $0.28/shot). Reviewing keyframes first lets you skip bad shots before spending.
  1. 1
    Orchestrator runs image generation
    For each scene: uses FLUX with character LoRA (if trained) + PuLID face injection. One image per scene. Progress shows in real-time — each shot checks in as it completes. Takes ~30s–2min per scene depending on queue.
  2. 2
    Dashboard shows active job
    The red pulsing banner on the Dashboard shows step + progress. You can leave this tab — come back and refresh.
// Step 5

Review & Approve Keyframes — Your Quality Gate

When keyframes finish, the scene list updates with thumbnails and toggle controls.

  1. 1
    Review each keyframe image
    Check face accuracy, composition, lighting. Each scene shows the generated image with an Approve/Skip toggle.
  2. 2
    Toggle shots you want animated
    Default is all approved. Click a shot to skip it (bad face, wrong composition). Use Select All / Deselect All for bulk control.
  3. 3
    Pick your animation model
    Set the model selector in the approve bar before clicking Animate.

    Free Wan 2.1 — fast, good for drafts and testing
    Free Wan 2.1 Best — slower, higher quality open model
    $0.28/shot Kling 2.6 Pro — production quality, use for final renders
// Step 6

Animate — Image to Video

Click ⚡ Animate Selected. This fires the /orchestrate/reanimate pipeline.

  1. 1
    I2V: Image → 5s video clip
    Each approved keyframe is sent to the selected model. Motion prompt is derived from the scene description. Output is a 5-second MP4 per shot.
  2. 2
    TTS: Dialogue → voice-over audio
    ElevenLabs generates the scene's dialogue using the character's cloned voice. Audio is saved alongside the clip.
  3. 3
    Timeline auto-updates
    Each completed shot is saved to the timeline as it finishes — no waiting for the whole job. Open Timeline ↗ at any point to see shots coming in live.
Don't close the backend tab during animation. The Railway server process runs the job — if it restarts mid-job you'll need to re-animate remaining shots.
// Step 7

Timeline — Arrange Your Film

Timeline ↗ (opens in new tab)
  1. 1
    Review shot order
    Shots load in scene index order. Drag cards to reorder. Click a card to preview the clip.
  2. 2
    Delete or replace bad shots
    Remove any shot you don't want in the final cut. Go back to Idea Vault → Keyframes to regenerate individual shots if needed.
  3. 3
    Save order
    Click Save Order to lock in your cut. This writes to MongoDB so it persists across sessions.
// Step 8

Export Film

In Timeline, click Export Film. The export modal opens.

  1. 1
    Set options
    Title Card — optional video file prepended as an opening title sequence
    Include Captions — burns dialogue text onto each clip
    Include Voice-Over — mixes TTS audio with each clip
    Music Track — background music from your Music library
  2. 2
    FFmpeg stitches the film
    Backend concatenates all clips in order, adds captions, mixes voice-over and music, outputs a single MP4. Takes ~1-3 minutes for a full film.
  3. 3
    Preview and download
    Video player appears in the export modal. Watch it, then click Download to save the MP4.

// Tools

Music

Go to the Music tab. Three options:

  • Upload Track — drag in any MP3/WAV. It goes into your library and is available in the export modal's music selector.
  • AI Generate — type a mood/style prompt, get a generated music track.
  • Mix — blend two tracks with a crossfade slider.
Tip: Music is mixed at 30% volume behind voice-over during export so dialogue stays audible.
// Tools

Image Gen — Manual Shot Creation

The Image Gen tab is for one-off image + animation work outside the main pipeline. Use it to:

  • • Regenerate a single bad keyframe with an edited prompt
  • • Test image styles before committing to a full story
  • • Generate standalone images for thumbnails, covers, or social posts
  • • Animate any existing image with a custom motion prompt
