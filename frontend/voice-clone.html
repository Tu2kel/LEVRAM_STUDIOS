How to Use Cloned Voices
ElevenLabs Clone Flow:
1. Upload an audio sample above → voice is registered in your ElevenLabs account.
2. The cloned voice ID is automatically available in the Voice Lab character select.
3. Generate TTS for any script line using the cloned voice.

RVC Flow:
1. Upload your .pth model above → it saves to data/rvc_models/.
2. In the Voice Lab, generate a raw TTS line, then hit "Voice FX" with RVC enabled.
3. The conversion runs locally using your uploaded model.

Assign to Character:
Go to Character Lab → select your character → set "ElevenLabs Voice ID" field to the ID shown in the library below.
